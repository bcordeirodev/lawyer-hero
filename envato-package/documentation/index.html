üé® Styling & Colors

Main Color Scheme

Edit src/app/globals.css to change the color scheme:

:root { --gold-500: #d4af37; /* Main accent color */ --gold-600: #b8941f; /* Darker accent */ /* Change these to your brand colors */ }

Dark/Light Mode

The template includes automatic dark/light mode. Colors are defined with CSS custom properties that automatically adapt.

Typography

The template uses Inter font from Google Fonts. To change fonts, edit the font imports in src/app/layout.tsx.

üìß Email Setup (Optional)

EmailJS Configuration

The contact form uses EmailJS for sending emails. To set it up:

1

Create EmailJS Account

Go to emailjs.com and create a free account.

2

Create Email Service

Add your email provider (Gmail, Outlook, etc.) in the EmailJS dashboard.

3

Create Email Template

Create a template with variables like {{client_name}}, {{client_email}}, etc.

4

Add Environment Variables

Create .env.local file in the src folder:

NEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key
‚úÖ Note: The form will work without EmailJS setup, but emails won't be sent. The lawyer's email in the config will still be displayed as contact information.

üöÄ Deployment

Vercel (Recommended)

  1. Push your code to GitHub
  2. Connect your GitHub repo to Vercel
  3. Vercel will automatically build and deploy
  4. Add environment variables in Vercel dashboard if using EmailJS

Netlify

  1. Connect your repo to Netlify
  2. Set build command: npm run build
  3. Set publish directory: out
  4. Add environment variables if needed

Traditional Hosting

For traditional hosting providers:

npm run build npm run export

Upload the generated out folder to your hosting provider.

üîß Troubleshooting

Common Issues

npm install fails

Solution: Update Node.js to version 18+ and try again.

Port 3000 already in use

Solution: Use a different port:

npm run dev -- -p 3001

Images not loading

Solution: Make sure images are in the public/images/ folder and paths are correct.

Email form not working

Solution: Check EmailJS configuration and environment variables.
